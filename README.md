# CSS-learning

# <center>Different Types of Selectors</center>

## 1. Name selector

```css
body{
    //css code
}
```

## 2. Class selector

```css
.body{
    //css code
}
```

## 3. ID selector

```css
#body{
    //css code
}
```

## 4. Decendent selector

```css
body h1{
    //css code
}
```

## 5. Child selector

```css
.body > p{
    //css code
}
```

</br>

## CSS Specificity

ascending order of CSS selectors

> * Inline selectors
> * ID selectors
> * Class/pseudo-classes/attribute selectors
> * Element/pseudo-elements
